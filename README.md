동네 복지관에서 영어 회화 재능기부를 하고 있는데,
막상 가르쳐보니 시중 영어 앱들이 저희 어르신들한테는 잘 안 맞더라고요.

그래서 딱 그것만 되는 앱을 만들었습니다.

- 문장 하나를 원하는 만큼 무한 반복 재생

- 광고 없음, 결제 없음, 완전 무료
- 개인정보 수집 안 함
