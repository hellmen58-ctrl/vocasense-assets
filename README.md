안녕하세요. 올해 69세, 코딩은 전혀 몰랐고 원래 중장비 일 하다가
은퇴한 사람입니다. 은퇴하고 심심해서 AI 도구 도움받아가며
앱 하나를 처음부터 끝까지 직접 만들어봤습니다.

영어 단어 학습용으로 시작했는데, 만들다 보니 구조 자체가
"뭐든 반복해서 외워야 하는 사람"한테는 다 통하겠더라고요.
학생 시험공부든, 자격증 암기든, 대사 외우기든 상관없이 쓸 수 있습니다.

시중 학습 앱들 보니 레벨 테스트니 결제니 광고니, 정작 필요한
"그냥 계속 반복해서 듣기"가 안 되는 경우가 많아서, 딱 그것만 되는
앱을 만들었습니다.

- 원하는 문장/단어를 무한 반복 재생
- AI 목소리로 자연스럽게 읽어줌 (인터넷 없이도 재생됨, 처음 받을 때만 필요)
- 화면 옆으로 밀면 목소리 바로 바뀌어서 발음 비교 가능
- 엑셀로 내가 외우고 싶은 내용 직접 넣어서 학습자료 제작 가능
- 광고 없음, 결제 없음, 완전 무료
- 개인정보 수집 안 함

나이도 전공도 상관없이 뭐든 만들 수 있는 시대가 됐다는 걸
직접 보여드리고 싶기도 했습니다.
