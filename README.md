# VocaSense (보카센스) — 누구나의 무한반복 암기 앱

**69세, 비전공, 은퇴한 중장비 기사가 AI 도구의 도움만으로 혼자 만든 영어 학습 앱입니다.**
코드 한 줄 몰랐던 사람이 직접 만든 앱이라니 믿기지 않으실 수도 있지만, 지금 여러분이 보고 계신 이 앱이 그 결과물입니다.

**단어든 문장이든, 외워야 하는 게 있다면 몇 번이든 반복해서 듣고 익히는 아주 단순한 암기 앱입니다.**
영어 단어, 시험 준비, 자격증, 대사 암기까지 — 화려한 커리큘럼이나 레벨업 시스템 없이, **진짜로 입에 붙을 때까지 반복**하는 것 하나에 집중했습니다.

---

## 이 앱이 왜 필요했나

시중의 학습 앱들은 "다음 레슨으로 넘어가라"고 재촉합니다. 광고를 봐야 하거나, 유료 결제를 유도하거나, 한 문장을 몇 번 이상 반복해서 듣는 것 자체를 앱이 막아두기도 합니다.

VocaSense는 그 반대입니다. **원하는 문장을 100번이든 몇 번이든, 원하는 만큼 무한 반복**해서 들을 수 있습니다. 개발자 본인이 직접 써보면서 "아는 건데도 입으로 안 나온다"는 문제가 이 방식으로 풀리는 걸 확인하고 만들었습니다. 영어 학습용으로 시작했지만, 구조 자체는 **외워야 할 게 있는 사람이라면 누구에게나** 그대로 유용합니다.

- **완전 무료**, 광고 없음
- 나이나 전공은 전혀 상관없다는 걸 몸소 증명하며 만든 앱입니다
- 학생, 수험생, 직장인, 어르신까지 — 반복 암기가 필요한 누구에게나 맞습니다
- AI 도구의 도움을 받아 비전공 독학으로 개발했습니다

## 주요 기능

- 🔊 **AI 음성**: 자연스러운 AI 목소리로 문장을 읽어줍니다 (인터넷 없이도 재생 가능, 처음 받을 때만 인터넷 필요)
- 🔁 **무한반복**: 버튼 하나로 같은 문장을 계속 반복 재생. 길게 누르면 반복 중간에 한글 뜻도 살짝 끼워 넣을 수 있음
- 👆 **스와이프로 목소리 전환**: 화면을 옆으로 밀면 다른 목소리로 즉시 바뀌어서 발음을 비교하며 들을 수 있음
- 📴 **오프라인 작동**: 음성파일만 받아두면 인터넷 연결 없이도 완벽하게 작동
- 🔋 **배터리 거의 안 씀**: 화면을 꺼둔 채로도 몇 시간이고 재생 가능
- 📚 **나만의 학습 자료 제작**: 엑셀 파일로 원하는 문장/단어를 직접 만들어 넣을 수 있음 (영어뿐 아니라 어떤 암기 자료든 가능)

## 다운로드 및 설치 방법

1. 이 저장소의 **[Releases](../../releases)** 페이지로 이동합니다
2. 최신 릴리스에서 **`vocasense.apk`** 파일을 다운로드합니다
3. 안드로이드 폰에서 다운로드한 파일을 눌러 설치합니다
   - "출처를 알 수 없는 앱" 경고가 뜨면 **설정 → 허용**을 눌러주세요 (구글 플레이스토어를 거치지 않고 직접 배포하는 앱이라 뜨는 정상적인 안내입니다)
4. 앱을 실행하면 **사용법을 읽어주는 첫 과목("user_manual")이 자동으로 열립니다**
5. 데이터관리 → 데이터받기 → **인터넷에서 받기**를 눌러 AI 음성 파일을 받으세요 (와이파이 환경 권장, 데이터를 사용합니다)

## 자주 묻는 질문

**Q. 왜 플레이스토어에는 없나요?**
A. 개인이 취미로 만든 무료 앱이라, 정식 스토어 등록 절차(테스터 모집, 심사 등) 없이 직접 배포하고 있습니다.

**Q. 정말 무료인가요?**
A. 네, 광고도 없고 결제도 없습니다.

**Q. 안전한가요?**
A. 이 앱은 여러분의 개인정보를 전혀 수집하지 않습니다.

**Q. 영어 말고 다른 것도 외울 수 있나요?**
A. 네. 엑셀로 원하는 문장/단어를 직접 넣을 수 있어서, 자격증 암기, 대사 연습 등 어디에나 쓸 수 있습니다.
